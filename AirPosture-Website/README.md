# AirPosture Website

This is the official website for AirPosture - an iOS app that turns your AirPods into a real-time posture coach.

Website: airposture.github.io

## Getting Started

**Setup**: `yarn install`

First, run the development server:

```bash
npm run dev
# or
yarn dev -- this one
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
